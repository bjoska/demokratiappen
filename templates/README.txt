This directory contains header-templates for the different source formats
that are used in Demokratiappen. They must be included as the first lines
of all produced source code files to be covered by the GPL license, see
https://www.gnu.org/licenses/gpl-howto.html for an introduction to the topic.

For C/C++/ObjectiveC/Java/JavaScript use header.h
For HTML/CSS/XML no license header is used.
